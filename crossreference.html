    1. Implement HTML5 structural elements (nav tag)
    2. Use of anchor references (services title)
    3. Links to appropriate resources, additional info ... pertaining to the nature of the business (maps link to external maps)
    4. User of hotspots
    5. Insert inline images and line breaks (past work page)
    6. Mark page headings, paragraphs, block quotes and addresses appropriately (heading and paragraphs are all over, address is in footer)
    7. Create appropriate ordered, unordered and descritption lists (this page has both ordered and unordered lists)
    8. Insert special characters from extended character sets (ampersand in "Winners & Home Sense")
    9. Implement meta tags (inside head tag of every page)
    10. Your home page must have a footer when the web site was last updated

  • 2.3.2 - Cascading Style Sheets Minimum Requirements Features

    1. Inline style on homepage heading
    2. Style homepage with embedded styles

    4. Use CSS3 to format the appearance of elements (All the pages)
    5. Use CSS3 style for color (all over style.css)
    6. Use CSS3 styles to format appearance of text (.jumpotron)
    7. Style hyperlink (no need for hover, visited, etc in design. basic style under nav li.active a)
    8. User of CSS3 background styles (body tag uses background-size: cover)
    9. CSS Dropcap (drop cap was not appropriate for design
    10. Define document content with class and id properties and create styles for them (all throughuot every page and the style sheet)
    11. Mark document content with class and id properties and create styles for them (all throughuot every page and the style sheet)
    12. Use styles to control page layout by using float attribute (.c2 and .c3 have the float attribute)
    13. Use a web font
